# Day 3 Notes

## Operators

Operators are symbols used to perform operations on variables and values.

---

## Arithmetic Operators

| Operator | Meaning |
|-----------|----------|
| + | Addition |
| - | Subtraction |
| * | Multiplication |
| / | Division |
| % | Modulus |
| ** | Exponent |
| // | Floor Division |

Example:

```python
10 + 5
```

---

## Comparison Operators

Used to compare values.

```python
==
!=
>
<
>=
<=
```

Example:

```python
10 > 5
```

Output:

```python
True
```

---

## Logical Operators

### and

Returns True if both conditions are True.

### or

Returns True if at least one condition is True.

### not

Reverses the result.

---

## Conditional Statements

### if

```python
if condition:
    print("True")
```

### if-else

```python
if condition:
    print("True")
else:
    print("False")
```

### if-elif-else

```python
if marks >= 90:
    print("A")
elif marks >= 80:
    print("B")
else:
    print("C")
```
